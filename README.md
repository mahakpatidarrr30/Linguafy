# 🌍 Linguafy

**Linguafy** is a modern mobile-first language learning app prototype designed to make vocabulary building fun, engaging, and visually immersive.It combines gamification, sleek UI design, and interactive learning flows inspired by modern EdTech platforms.



Linguafy helps users practice languages through:

🃏 Interactive flashcards  
🧠 Quizzes with XP rewards  
📚 Searchable vocabulary library  
🏆 Competitive leaderboard  
👤 Gamified profile progress tracking
